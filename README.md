# Dress Redesign Agent

An AI-powered tool that redesigns a dress from a photo. Upload an image, fill in your style preferences, and receive a fashion sketch + step-by-step seamstress instructions — all generated by a multi-agent pipeline built on Google ADK and Gemini.

## What it does

1. **Trend Research** — searches Google for current fashion trends matching the chosen style
2. **Dress Analysis** — analyzes the uploaded photo using Gemini multimodal vision
3. **Design Creation** — generates a redesign concept grounded in trend research and user preferences
4. **Design Validation** — checks the concept for structural feasibility, consistency, and form compliance before any image is generated
5. **Sketch Generation** — produces a front + back fashion sketch using Gemini image generation
6. **Seamstress Guide** — writes detailed construction instructions tied to the specific design

## Architecture

```
SequentialAgent (DressRedesignPipeline)
├── TrendResearchAgent     — google_search tool, outputs trend_insights
├── DressAnalyzerAgent     — native multimodal, outputs dress_analysis
├── DesignCreatorAgent     — outputs design_concept
├── DesignValidatorAgent   — validates + corrects, outputs final_design_concept
├── ImageGeneratorAgent    — calls generate_dress_sketch tool
└── SeamstressGuideAgent   — outputs seamstress_guide
```

Each agent reads from and writes to a shared session state via `output_key`. Agents use `include_contents="none"` to prevent conversation history leakage between steps.

## Tech stack

- [Google ADK](https://google.github.io/adk-docs/) 2.3.0 — multi-agent orchestration
- [Gemini 2.5 Flash](https://deepmind.google/technologies/gemini/) — validation (reasoning-heavy)
- [Gemini Flash Lite](https://deepmind.google/technologies/gemini/) — trend research, analysis, design, seamstress guide
- [Gemini 3.1 Flash Image](https://deepmind.google/technologies/gemini/) — sketch generation
- [Streamlit](https://streamlit.io/) — web UI

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Create a `.env` file in the project root:

```
GOOGLE_API_KEY=your_api_key_here
```

## Run

```bash
streamlit run app.py
```

## Test

```bash
pytest tests/ -v
```
